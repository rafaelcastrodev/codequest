# Melhorias

- Perfil deve permitir edição do nome do usuário. Atualmente é travado em `Jogador`.
- Durante a lição, não há um botão explícito para encerrar a lição.
- Desative o auto-complete do editor;
- Nível inicial atualmente é 0. O correto é 1.
- Renomear `Badges` para `Troféus`.
- Adicionar Menu/Página `Sobre`. Explicando o que é, como usar e quem fez o aplicativo.
- Renomear `Streak` para `Ofensiva`.
- Menu item `Mapa` vira `Jornada`.
- Sistema de vidas é opcional e desligado por default.
- Caso sistema de vidas esteja `desligado` os indícios de vida (corações, etc) devem ser escondidos da UI.
